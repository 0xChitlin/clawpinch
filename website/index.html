Get Started

Three commands. That is all it takes.

Install

# Clone the repository
$ git clone https://github.com/clawpinch/clawpinch.git
$ cd clawpinch

# Run the audit
$ bash clawpinch.sh

# Or add as an OpenClaw skill
$ openclaw skill add ./clawpinch
$ openclaw skill run clawpinch

Requires bash >= 4.0 and jq. Optional: openssl, nmap, sha256sum.
